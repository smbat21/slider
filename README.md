# Slider

### Запуск 
```js
slider({
    container: 'блок содержащий слайдер',
    slide: 'каждый слайд',
    field: 'Обретка slide. Полоса из всех слайдов',
    wrapper: 'Обретка field. Окно, через ширину которого, отображается текущий слайд',
    nextArrow: 'пролистать вправо',
    prevArrow: 'пролистать влево',
    totalCounter: 'количество слайдов',
    currentCounter: 'номер текущего слада'
});
```
